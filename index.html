Wörterbuch

German Dictionary Made Easy

Download this project as a .zip file Download this project as a tar.gz file

Wörterbuch is a Google Chrome extension that makes looking up German words in The Free Dictionary much easier.

Usage

  • On any page highlight a word, right click on it, select Wörterbuch from the menu.
  • Or go to the omnibox (URL bar), type "de", then space and any word to look up.

Extras

The extension tries to make The Free Dictionary more readable by adjusting fonts, spacings and removing unnecessary content. This might have side-effects but shouldn't affect the basic functionality (define the meaning of words:)
